รอจนหน้าจอขึ้นคำว่า Ready (สีฟ้า) จากนั้นใช้ปุ่มสีดำด้านข้างอุปกรณ์เพื่อสั่งงาน:

👆 กด 1 ครั้ง (Click)

เลื่อนเมนู (Next)

กดเพื่อเลื่อนลูกศรสีเขียวไปยังเมนูถัดไป

⏱️ กดค้าง 5 วินาที (Hold)

เลือกเมนู (Select)

กดแช่ไว้จนขึ้นคำว่า SELECTED! แล้วปล่อยนิ้ว

📋 คำอธิบายเมนูหน้าจอ

ชื่อเมนูบนจอ หน้าที่และการทำงาน
1. Enable Interfaces เปิดพอร์ตฮาร์ดแวร์ที่จำเป็น
เปิด SSH, VNC, SPI, I2C, 1-Wire, Hardware Serial
ปิด Serial Console (ป้องกัน Log กวนระบบ Modbus)
2. Install VNC ติดตั้งระบบ BossHub VNC ทำงานอยู่เบื้องหลัง
3. Install Coin Machine ติดตั้งแอปพลิเคชันตู้แลกเหรียญ ทำงานอยู่เบื้องหลัง
4. Install ALL รันการติดตั้งข้อ 1, 2 และ 3 ต่อเนื่องกันรวดเดียวจบ
5. Macro: Y & Password เมนูแก้ปัญหาเฉพาะหน้า: หากจอค้างถาม [Y/n] หรือถามรหัสผ่าน ให้กดเมนูนี้ ระบบจะพิมพ์ y และรหัส 123456 ให้ทันที
6. Open Terminal (CMD) เปิดหน้าต่าง Terminal ขึ้นมาใหม่บนหน้าจอ
7. Reboot System สั่งรีสตาร์ทเครื่อง Raspberry Pi
